# open-mo

会話の中で直前に読み書き・編集したMarkdownファイル（.md）を特定し、`mo` コマンドで開く。

## 手順

1. 会話のコンテキストから、直近で扱った `.md` ファイルのパスを特定する。複数ある場合はすべて対象とする。
2. 対象ファイルが存在することを確認する。
3. 以下のコマンドを実行する:

```
mo <対象ファイルパス...>
```

- 複数ファイルがある場合はスペース区切りで渡す（例: `mo file1.md file2.md`）
- ユーザーが引数で特定のファイルやオプション（`--target` など）を指定した場合はそれに従う

## 注意

- Markdownファイルが会話中に見つからない場合は、ユーザーにどのファイルを開くか確認する
- `mo` コマンドはバックグラウンドでサーバーを起動するため、`run_in_background` で実行する
